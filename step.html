Step of Uploading

I.Upload chapter (Old 1-6 New 1-5)

1. Get Chapter. Open website that contrains chapter and Use Chrome extension named "Download All Images". Click "Gallery", Change name of file (NOTE: Name must be in pattern Xx000p000.jpg only eq.Hd070p0[extension].jpg, H = firstcharracter d = lastcharracter 070 = chapter 70 p = page 000 = count page) and Click Download. Download all chapter in one step eq.chapter1-10 and go to step 2.



2. UnZip files. Use java code in Manhtia(II). Choose all Zip files and run program it will altomatically unzip and delete Readme.txt. All unzipped folder will stored in Download/Unzip.



3. Create google drive folder. Copy ID of parent folder (Name of story will upload to.) Eq. Create folder 11-46 Start = 11 Stop = 47 Id = Id of folder that folders will create in.



4. Upload Chapters. Type ID of parent folder. Click Search Data will appear and click new for choose files to upload. Input box is search chapter, Select box is chapter will upload to. Choose files Max files = 60 files(Mannually) 40-50(Auto). Checkbox to skip(Auto mode). For use auto mode check Auto below and Press Confirm(After already Choose chapter and files). ***Note Auto mode might cause sn error, After finish Please check upload error(Click link) search by parent id.


5.1.(NEW Reccomand) Append link. Paste Parent folder ID, Chapter name and Drop files that system will upload to. Press search. And tick the chapter that want to Upload by range or manually. For Auto check Auto and press confirm(Suggest).


5.2.(NEW) Append link. Paste Parent folder ID and choose name that system will upload to. Press search. And tick the chapter that want to Upload by range or manually. For Auto check Auto and press confirm(Suggest).


5.3.(New) Append link. Paste folder that contrains files ID, choose Chapter, and Name. Type Chapter count and Status(If has no status type 'n') Press only submit.

5.4.(NEW) Append link. Paste folder that contrains files ID, choose Chapter Press only submit. After this will download Finale.csv


5.(OLD) Sort link. Click Sort link, Choose file, Copy Paste to Raw data csv file from Change data in ManhwaApp.


6.(OLD) In finale.csv file copy second column and paste it in each story file. Info2 of caption must be yes.
